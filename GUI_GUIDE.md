# Auto Recorder GUI 版本说明

## 🎨 新功能：图形界面

v0.2.0 新增了完整的图形用户界面（GUI），让录音更加直观便捷！

## 📦 两个版本

### 1. CLI 版本（命令行）
- `auto-recorder.exe` - 命令行工具
- 适合：高级用户、脚本自动化、后台运行
- 使用方式：命令提示符或 `launcher.bat`

### 2. GUI 版本（图形界面）⭐ 新增
- `auto-recorder-gui.exe` - 图形界面程序
- 适合：普通用户、日常使用
- 使用方式：双击直接运行

## 🚀 GUI 版本快速开始

### 方法 1：直接运行（推荐）

1. 双击 `auto-recorder-gui.exe`
2. 图形界面自动打开
3. 点击"开始"按钮即可录音

### 方法 2：使用启动脚本

双击 `start_gui.bat`

## 🎯 GUI 界面功能

### 主界面

```
┌─────────────────────────────────────────┐
│  00:00:00.0   [开始] [停止]   [查看]   │
│                                         │
│  当前模式：语音视频聊天自动录音。       │
│                                         │
│  🎤 ████████████░░░░░░░░░░░░           │
│  🔊 ██████████████░░░░░░░░░░           │
│                                         │
│  ⚙ 设置                                │
└─────────────────────────────────────────┘
```

#### 功能说明

1. **计时器显示**
   - 显示录音时长：HH:MM:SS.D
   - 实时更新

2. **控制按钮**
   - `开始` - 开始录音
   - `停止` - 停止录音并保存
   - `查看` - 打开录音文件夹

3. **模式提示**
   - 显示当前录音模式
   - 根据设置自动更新

4. **音频电平指示器**
   - 🎤 麦克风输入音量（绿色到红色渐变）
   - 🔊 扬声器输出音量（绿色到红色渐变）
   - 实时显示音频活动

5. **设置按钮**
   - 点击打开详细设置窗口

### 设置窗口

点击主界面的"⚙ 设置"按钮打开设置窗口：

#### 录音模式

##### 1. 手动录音
- 完全手动控制
- 点击开始录音，点击停止结束

##### 2. 软件启动后自动录音 ⭐
- 程序启动即开始录音
- 设置自动保存间隔（分钟）
- 静音检测阈值（百分比）

##### 3. 当如下程序开始语音/视频聊天时自动开始录音
- 监控指定的应用程序
- 检测到通话自动录音
- 通话结束自动停止
- 支持的应用：
  ```
  QQ.exe, Skype.exe, WeChat.exe, Weixin.exe, 
  WhatsApp.exe, WXWork.exe, Teams.exe, Zoom.exe
  ```

##### 4. 计划录音模式
- 按时间表自动录音
- 设置开始时间和录音时长
- 支持多个计划任务

#### 音频源设置

- ○ 录制从麦克风输入的声音
- ○ 录制从电脑播放的声音
- ● 输入及播放的声音均进行录制 ✓
  - ☐ 声音保存在不同声道中

#### 保存设置

1. **保存路径**
   - 默认：`D:\Documents\录音`
   - 点击"浏览"选择其他位置

2. **文件夹组织**
   - ☐ 创建月份文件夹（如：2024-02）
   - ☐ 创建日期文件夹（如：2024-02-07）

3. **音频参数**
   - 采样频率：8000 / 11025 / 22050 / 44100 / **48000** Hz
   - 比特率：32 / 64 / 96 / **128** / 192 / 256 / 320 kbps

## 📋 使用场景

### 场景 1：日常会议录音
1. 打开 GUI 程序
2. 选择"软件启动后自动录音"
3. 设置自动保存为 60 分钟
4. 最小化程序，开始会议
5. 会议结束后点击"查看"获取录音

### 场景 2：监控通话应用
1. 打开 GUI 程序
2. 点击"设置"
3. 选择"当如下程序开始语音/视频聊天时自动开始录音"
4. 确认监控列表包含你使用的应用
5. 关闭设置，程序开始监控
6. 微信/QQ 通话时自动录音

### 场景 3：定时录音
1. 打开 GUI 程序
2. 点击"设置"
3. 选择"计划录音模式"
4. 添加录音计划（如每天 9:00-17:00）
5. 程序按计划自动录音

## 🐛 常见问题

### Q: GUI 程序无法启动
**A:** 确保安装了 Visual C++ Redistributable:
- [下载链接](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### Q: 界面显示乱码
**A:** Windows 系统语言设置为中文，或等待后续版本字体优化

### Q: 音频电平条不动
**A:** 检查音频设备是否正常工作，启用立体声混音

### Q: 设置窗口太大/太小
**A:** 可以拖动窗口边缘调整大小，设置会自动保存
