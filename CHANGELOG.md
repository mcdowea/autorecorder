# 更新日志 (Changelog)

所有值得注意的项目更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/),
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2026-02-08

### 新增
- 🎙️ **手动录音功能**: 添加独立的手动录音按钮,不依赖麦克风检测
- 🎵 **扩展采样率选项**: 新增 16kHz, 22.05kHz, 96kHz 采样率选项
- 💿 **更多比特率选项**: 新增 64, 320 kbps 比特率选项
- ⚙️ **MP3 质量控制**: 可调节 LAME 编码质量 (0-9 级别)
- 🤖 **GitHub Actions**: 自动编译和发布工作流
- 📝 **配置文档**: 添加详细的配置文件说明 (CONFIG.md)

### 改进
- ✨ 优化 GUI 界面布局,添加手动录音状态显示
- 📊 改进音质设置界面,提供更直观的选项说明
- 🔧 配置文件新增 `mp3_quality` 参数

### 文档
- 📖 更新 README.md,补充新功能说明
- 📋 新增 CONFIG.md 配置文件详细说明
- 📝 新增 CHANGELOG.md 版本更新记录

## [1.0.0] - 2026-02-01

### 新增
- 🎤 **智能麦克风检测**: 自动检测系统麦克风使用状态
- 🎧 **双通道录音**: 同时录制麦克风和扬声器音频
- 🔊 **实时混音**: 自动混合双通道音频
- 💾 **MP3/WAV 编码**: 支持 MP3 和 WAV 两种格式
- 🎛️ **音量增益控制**: 独立调节麦克风和扬声器增益
- 🚫 **进程黑名单**: 可配置不触发录音的应用程序
- 📁 **智能文件管理**: 按应用名自动创建文件夹
- 📊 **实时电平监控**: 可视化音频电平显示
- ⚙️ **图形化配置界面**: Windows GUI 应用
- 🖥️ **命令行版本**: 支持命令行运行

### 技术特性
- Windows Core Audio API 集成
- WASAPI Loopback 音频捕获
- LAME MP3 编码器
- 多线程音频处理
- 实时音频流处理

## 版本说明

### 版本号规则

本项目采用语义化版本号: `主版本号.次版本号.修订号`

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 发布频率

- 重大功能更新: 每 1-2 个月
- 功能改进版本: 每 2-4 周
- 问题修复版本: 根据需要随时发布

### 获取最新版本

- **Releases 页面**: https://github.com/your-username/smart-recorder/releases
- **自动更新**: 未来版本将支持应用内更新检查
