# 🚀 快速入门指南

## 安装和首次运行

### 1. 下载程序

从 [Releases 页面](https://github.com/your-username/smart-recorder/releases) 下载最新版本的 `smart-recorder-windows-vX.X.X.zip` 文件。

### 2. 解压文件

将 ZIP 文件解压到任意目录,例如:
```
C:\Program Files\SmartRecorder\
```

解压后的文件结构:
```
SmartRecorder/
├── smart-recorder.exe         # 命令行版本
├── smart-recorder-gui.exe     # 图形界面版本 (推荐)
├── README.md                  # 项目说明
└── config.example.json        # 配置示例
```

### 3. 首次运行

双击运行 `smart-recorder-gui.exe`

首次运行时:
- 程序会自动创建配置文件 `smart_recorder_config.json`
- 默认录音保存路径为 `D:\Recordings`
- 可以在设置中修改保存路径

## 基本使用

### 自动录音模式

1. 点击 **"▶️ 开始监控"** 按钮
2. 程序开始监控麦克风使用情况
3. 当检测到应用使用麦克风时,自动开始录音
4. 麦克风不再使用时,自动停止并保存录音
5. 录音文件保存在配置的目录中

### 手动录音模式

1. 点击 **"🎙️ 手动录音"** 按钮
2. 开始录制所有音频 (麦克风 + 扬声器)
3. 再次点击 **"⏹️ 停止录音"** 结束录制
4. 录音文件以 `manual_` 前缀保存

### 查看录音文件

- 点击 **"📁 打开文件夹"** 按钮
- 或手动打开配置的保存路径
- 录音文件按应用名称分文件夹保存 (可在设置中关闭)

## 常用设置

### 1. 修改保存路径

**设置 → 基本设置 → 保存路径**

点击"浏览"按钮选择新的保存位置。

### 2. 调整音频质量

**设置 → 音质设置**

推荐配置:
- **日常使用**: 48000 Hz, 128 kbps, MP3 格式, 质量 2
- **高质量**: 48000 Hz, 192 kbps, MP3 格式, 质量 2
- **专业录音**: 96000 Hz, WAV 格式

### 3. 调节音量平衡

**设置 → 音量增益**

如果对方声音太小:
- 提高"扬声器增益"(例如调到 1.5x)

如果自己声音太小:
- 提高"麦克风增益"(例如调到 1.5x)

### 4. 配置黑名单

**设置 → 进程黑名单**

添加不希望触发录音的应用,例如:
```
chrome.exe,firefox.exe,msedge.exe,Discord.exe
```

多个程序用英文逗号分隔。

### 5. 设置最小录音时长

**设置 → 其他设置 → 最小录音时长**

建议设置为 3-5 秒,避免误触发产生大量短录音。

## 实时监控

### 音频电平指示

主界面显示实时音频电平:
- **麦克风电平**: 显示您的麦克风音量
- **扬声器电平**: 显示对方的扬声器音量

电平条动态显示音频强度,帮助您:
- 确认麦克风正常工作
- 调整合适的增益设置

### 运行日志

主界面底部的日志窗口显示:
- 监控状态变化
- 录音开始/停止
- 文件保存信息
- 错误提示

## 常见使用场景

### 场景 1: 录制语音通话

**目标**: 自动录制 QQ、微信、Zoom 等应用的通话

**设置**:
1. 开启"开始监控"
2. 确保通话应用不在黑名单中
3. 进行通话,程序会自动录制
4. 通话结束后自动保存

**推荐配置**:
- 采样率: 48000 Hz
- 比特率: 128 kbps
- 格式: MP3
- 质量: 2

### 场景 2: 录制在线课程

**目标**: 录制完整的在线课程音频

**操作**:
1. 课程开始前点击"手动录音"
2. 课程进行中持续录制
3. 课程结束后点击"停止录音"

**推荐配置**:
- 采样率: 48000 Hz
- 比特率: 192 kbps (课程内容重要)
- 格式: MP3
- 质量: 2

### 场景 3: 录制音乐或播客

**目标**: 高质量录制音乐或播客

**操作**:
使用手动录音功能

**推荐配置**:
- 采样率: 96000 Hz (超高质量)
- 格式: WAV (无损)
- 或: 比特率 320 kbps, MP3, 质量 0

### 场景 4: 快速语音备忘

**目标**: 快速录制语音备忘录

**操作**:
1. 快速点击"手动录音"
2. 说话
3. 点击"停止录音"

**推荐配置**:
- 采样率: 22050 Hz (节省空间)
- 比特率: 96 kbps
- 格式: MP3
- 质量: 5 (快速编码)

## 故障排除

### 问题 1: 检测不到麦克风使用

**可能原因**:
- 应用在黑名单中
- 麦克风权限问题

**解决方法**:
1. 检查"进程黑名单"设置
2. 确保应用有麦克风权限
3. 查看运行日志的错误信息

### 问题 2: 只能录到一方声音

**可能原因**:
- 某一通道的增益设置为 0
- 音频设备问题

**解决方法**:
1. 检查"麦克风增益"和"扬声器增益"都不为 0
2. 观察实时音频电平,确认两个通道都有信号
3. 测试系统的麦克风和扬声器是否正常

### 问题 3: 录音文件很大

**解决方法**:
1. 使用 MP3 格式而非 WAV
2. 降低采样率 (例如 44100 或 22050 Hz)
3. 降低比特率 (例如 96 或 128 kbps)
4. 提高 MP3 质量等级 (例如 5 或 7)

### 问题 4: 录音音质不好

**解决方法**:
1. 提高采样率到 48000 Hz 或更高
2. 提高比特率到 192 或 256 kbps
3. 降低 MP3 质量等级到 2 或 0
4. 考虑使用 WAV 格式

## 高级技巧

### 技巧 1: 批量处理录音

录音文件按应用名称自动分类到不同文件夹,便于管理:
```
Recordings/
├── QQ/          # QQ 的所有录音
├── WeChat/      # 微信的所有录音
└── Zoom/        # Zoom 的所有录音
```

### 技巧 2: 定期清理短录音

设置合适的"最小录音时长"(3-5秒),自动过滤极短的误触发录音。

### 技巧 3: 不同场景使用不同配置

可以手动编辑 `smart_recorder_config.json` 创建多个配置文件:
- `config_highquality.json` - 高质量配置
- `config_lowspace.json` - 节省空间配置

需要时复制对应配置文件为 `smart_recorder_config.json` 使用。

### 技巧 4: 命令行使用

对于高级用户,可以使用命令行版本:
```bash
smart-recorder.exe --output-dir "D:\MyRecordings" --sample-rate 48000
```

运行 `smart-recorder.exe --help` 查看所有命令行选项。
