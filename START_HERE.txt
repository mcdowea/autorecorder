⚠️  重要提示
═══════════════════════════════════════════════════════════════

如果您双击 auto-recorder.exe 或 auto-recorder-windows-x64.exe
没有任何反应，这是 ** 正常现象 **！

这是一个命令行程序，需要通过特殊方式运行。


🚀 快速开始（3个步骤）
═══════════════════════════════════════════════════════════════

第 1 步：双击运行
───────────────────────────────────────────────────────────
    📂 launcher.bat        ← 双击这个！
───────────────────────────────────────────────────────────

第 2 步：启用立体声混音（录制扬声器必需）
───────────────────────────────────────────────────────────
    1. 右键点击音量图标 → 声音设置
    2. 更多声音设置 → 录制标签
    3. 右键空白处 → 显示已禁用的设备
    4. 找到"立体声混音" → 右键启用
───────────────────────────────────────────────────────────

第 3 步：开始使用
───────────────────────────────────────────────────────────
    在 launcher.bat 菜单中：
    - 选择 [1] 生成配置文件
    - 选择 [3] 开始录音
───────────────────────────────────────────────────────────

═══════════════════════════════════════════════════════════

🔧 遇到问题？
═══════════════════════════════════════════════════════════

问题 1: 双击 .exe 没反应
───────────────────────────────────────────────────────────
解决：请使用 launcher.bat，不要直接双击 .exe
───────────────────────────────────────────────────────────

问题 2: 提示缺少 DLL 或 VC++ 运行时
───────────────────────────────────────────────────────────
解决：安装 Visual C++ Redistributable
      https://aka.ms/vs/17/release/vc_redist.x64.exe
───────────────────────────────────────────────────────────

问题 3: 杀毒软件报警
───────────────────────────────────────────────────────────
解决：添加程序文件夹到白名单（这是误报）
───────────────────────────────────────────────────────────

问题 4: 其他问题
───────────────────────────────────────────────────────────
解决：双击运行 diagnose.bat 进行诊断
───────────────────────────────────────────────────────────


💡 使用技巧
═══════════════════════════════════════════════════════════

✓ 使用 launcher.bat 最方便（图形菜单）
✓ 录音文件保存在 recordings\ 文件夹
✓ WAV 文件很大，可以转换为 MP3 节省空间
✓ 自动录音模式会监控微信、QQ、Teams 等应用
✓ 遇到问题先运行 diagnose.bat 诊断
